01. Ages
This function determines whether based on the given age a person is: baby, child, teenager, adult, elder. The input comes as single number 
parameter. 

02. Rounding
This function rounds numbers to specific precision. The input comes as two numbers. The first value is the number to be rounded and the 
second is the precision (significant decimal places). If a precision is passed, that is more than 15 it should automatically be reduced to 
15.Remove trailing zeroes, if any.

03. Division
This function receives a number as parameter and returns whether that number is divisible by the following numbers: 2, 3, 6, 7, and 10. 
It should always take the bigger division.
