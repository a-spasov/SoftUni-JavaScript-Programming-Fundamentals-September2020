01. Multiply Number by 2
This function receives a number and prints as result that number multiplied by two.

02. Excellent Grade
This function receives a single number(grade) and checks if the grade is excellent or not.

03. Numbers from 1 to 5
This function prints all the numbers from 1 to 5 (inclusive) each on a separate line.

04. Numbers from N to 1
This function receives a number N and prints all the numbers from N to 1. 

05. Numbers from M to N
This function receives a number M and a number N (M will always be bigger than N) and prints all numbers from M to N.

07. Month Printer
This function takes an integer from the console and prints the corresponding month. If the number is more than 12 or less than 1 print 
"Error!".

08. Foreign Languages
This function prints the language, that a given country speaks. You can receive only the following combinations: English is spoken in 
England and USA; Spanish is spoken in Spain, Argentina and Mexico; for the others, function should print "unknown".

09. Theatre Promotions
This function calculates the price of a single ticket. If the given age does not fit one of the categories, function should print "Error!".

10. Divisible by 3
This function prints all the numbers from 1 to 100, which are divisible by 3. You have to use a single for loop. The program should not 
receive input.

11. Sum of Odd Numbers
This function receives a number n as parameter and prints the next n odd numbers (starting from 1) and on the last row prints the sum of 
them.
