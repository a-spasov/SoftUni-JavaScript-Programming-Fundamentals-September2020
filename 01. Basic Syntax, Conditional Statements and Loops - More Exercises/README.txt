01. Sort Numbers
This function receives three numbers and sorts them in descending order. 

02. English Name of the Last Digit	
This function returns the English name of the last digit of a given number. 

03. Next Day
This function calculates the date of the next day by given year, month and day. The input comes as three number parameters. The first element 
is the year, the second is the month and the third is the day.

04. Reverse String
This function reverses a string and print it on the console.

05. Distance Between Points
This function calculates the distance between two points by given x and y coordinates.
