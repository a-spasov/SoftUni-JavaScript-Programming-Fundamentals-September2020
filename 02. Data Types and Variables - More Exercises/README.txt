01. Digits with Words
This function receives a digit in the form of a word as string and prints the digit as a number.

02. Prime Number Checker
This function checks if a number is prime.

03. Cone
This function calculates a cone’s volume and total surface area by given height and radius at the base.

04. Biggest of 3 Numbers
This function finds the biggest of 3 numbers.

05. Binary to Decimal
This function reads an 8-bit binary number and converts it to a decimal.

06. Chess Board
This function prints a chessboard of size n X n. The input comes as a single number argument n.
The output should be returned in the form of a string.

07. Triangle Area
This function calculates a triangle’s area by its 3 sides.
