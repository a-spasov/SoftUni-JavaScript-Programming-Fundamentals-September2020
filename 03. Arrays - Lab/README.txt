01. Sum First and Last Array Elements
This function receives an array of numbers and prints the sum of first and last element in that array.

02. Day of Week
This function receives a number and prints the corresponding name of the day of week. 
If the number is NOT a valid day, print 'Invalid day!'.

03. Reverse an Array of Numbers
This function receives a number n and an array of elements. It creates a new array with first n numbers of the given array, reverses it
and prints its elements on a single line, space separated.

04. Reverse an Array of Strings
This function receives an array of strings (space separated values) and swaps them.

05. Sum Even Numbers
This function receives an array of strings, parses them to numbers and sums only the even numbers.

06. Even and Odd Subtraction
This function calculates the difference between the sum of the even and the sum of the odd numbers in an array.

07. Equal Arrays
This function receives two string arrays and prints on the console whether they are identical or NOT. Arrays are identical if their 
elements are equal.  

08. Condense Array to Number
This function receives an array of numbers and condenses them by summing adjacent couples of elements until a single number is obtained.
