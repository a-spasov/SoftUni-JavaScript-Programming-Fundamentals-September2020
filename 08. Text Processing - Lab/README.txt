01. Print Characters
Write a function that receives a string and prints all the characters on separate lines.

02. Substring
Write a function that receives a string and two numbers. The numbers will be a starting index and count of elements 
to substring. 

03. Censored Words
Write a function that receives a text as a first parameter  and a single word as a second. Find all occurrences of 
that word in the text and replace them with the corresponding count of '*'.

04. Count String Occurrences
Write a function that receives a text and a string that you need to search. Print all the occurrences of that word 
in the string.
